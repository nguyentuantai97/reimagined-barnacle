## Cách lưu ảnh

Sau khi tạo ảnh, lưu vào thư mục:
```
public/images/products/
```

Với tên file theo format:
```
tra-sua.jpg
tra-sua-tc-trang.jpg
tra-sua-socola.jpg
tra-xanh-xoai.jpg
...
```

## Kích thước đề xuất
- **Kích thước**: 800x800px hoặc 1000x1000px (vuông)
- **Format**: JPG hoặc WebP
- **Chất lượng**: 80-90%

## Lưu ý quan trọng
1. **KHÔNG** thêm trái cây tươi vào ảnh (vì quán pha từ syrup)
2. Giữ background đơn giản (trắng hoặc gradient nhẹ)
3. Đảm bảo logo AN có thể nhìn thấy được
4. Ảnh nên có ánh sáng đẹp, chuyên nghiệp
